## Code

```html
<a-breadcrumb icon="path/to/icon.svg">...</a-breadcrumb>
```

```html
<a-breadcrumb icon="src/web-components-toolbox/src/icons/chevron_right.svg">
	<ul>
	    <li><a href="#">Home</a></li>
	    <li><span></span></li>
	    <li><a href="#">Item</a></li>
	    <li><span></span></li>
	    <li><a href="#">Lorem</a></li>
  	</ul>
</a-breadcrumb>
```
